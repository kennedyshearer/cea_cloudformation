# AWS CloudFormation Architecture Practice

## Overview

This repository contains hands-on AWS CloudFormation projects where I practice designing and deploying cloud architectures using Infrastructure as Code (IaC).

The goal of this repository is to strengthen my understanding of AWS services, architecture patterns, and automated infrastructure provisioning by building solutions to different cloud scenarios using CloudFormation templates.

Rather than manually creating resources through the AWS Console, each solution focuses on defining AWS infrastructure through reusable YAML templates that can be deployed, modified, and version-controlled.


## Skills Practiced

- AWS CloudFormation
- Infrastructure as Code (IaC)
- YAML template development
- AWS resource provisioning
- Cloud architecture design
- IAM security principles
- Networking fundamentals
- AWS service integration
- Resource dependencies and outputs
